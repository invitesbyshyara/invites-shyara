## What Is Shyara?

Shyara (invite.shyara.co.in) is a digital invitation SaaS platform. Customers create a
beautiful, animated, interactive invitation webpage and share it as a single link —
instead of a printed card or a flat JPEG image.

**How it works:**
1. Customer picks a template
2. Customizes it with names, dates, venue, photos, love story, event schedule
3. Gets a shareable URL (e.g., invite.shyara.co.in/i/sarah-and-ryan)
4. Shares that link via WhatsApp, Instagram, email, or text
5. Guests open the link, see a beautiful animated cover they "tap to open," scroll
   through the full invitation, and RSVP directly inside it

**Company:** Based in India. Accepts USD and EUR from international customers.
**Payment model:** One-time purchase per template (not a subscription).

## Product Features

### For the person creating the invite

**1. Animated opening cover**
5 animation styles per template: doors opening, curtain reveal, zoom-burst, fade-up,
slide-left. Guests tap/click to open — feels like unwrapping a gift.

**2. Modular sections (toggle on/off per invite)**
- **Hero** — Names, date, time (always shown, cannot be removed)
- **Love Story / Event Description** — Optional written story or event background
- **Venue & Location** — Venue name, address, map
- **Event Schedule / Timeline** — Time-ordered events list (e.g. "6:00 PM — Ceremony")
- **Photo Gallery** — Up to 10 photos
- **RSVP Form** — Guests respond directly inside the invite

**3. 4-step guided wizard**
- Step 1: Event details (names, date, time)
- Step 2: Venue + story/description
- Step 3: Gallery photos + event schedule
- Step 4: Live preview + choose custom URL + publish

**4. Live preview while editing**
See exactly how the invite looks (toggle between mobile and desktop view) in real time
as you fill the form.

**5. Custom shareable URL**
Examples: `/i/sarah-and-ryan`, `/i/john-turns-30`, `/i/priya-weds-arjun`

**6. Draft saving**
Save progress, come back and finish later.

**7. Edit after publishing**
Update venue, time, or any details anytime — all guests automatically see the update
because it's a live URL, not a sent image.

### For guests receiving the invite

- Tap/click animated cover to open
- Scroll through a fully interactive, mobile-optimized webpage
- RSVP directly inside (name, yes/no/maybe, guest count, personal message)
- Works on any device — no app download needed
- Instant thank-you screen after RSVPing

### For the host — after publishing

- RSVP dashboard: see who's coming, declined, or said maybe
- Guest count totals
- Guest messages
- Real-time updates as new RSVPs come in

## Competitive Landscape

| Competitor | Problem |
|---|---|
| **Evite** | Old, ad-heavy, ugly design, feels like 2005 |
| **Paperless Post** | Beautiful but subscription-based, expensive, US-centric |
| **Zola** | Wedding-only, very US-focused, not for other event types |
| **Canva invitations** | Static image or PDF — no interactivity, no RSVP |
| **WhatsApp JPEG** | Flat image, no RSVP, no editing, no analytics |
| **Printed invites** | $300–$1,000+ for weddings, wasteful, no RSVP |

### Shyara's differentiators
1. **Interactive animated webpage** — not a flat image or PDF
2. **Built-in RSVP** works directly inside the invite link
3. **One-time pricing** at $2.99–$7.99 — no subscription
4. **Works for ALL occasions** — not just weddings
5. **Mobile-first** — WhatsApp-shareable, no app needed
6. **Free tier** — zero friction to start, no credit card
7. **Edit after sending** — update details anytime (live URL)
